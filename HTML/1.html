HTML이란 무엇인가?

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications. Web browsers receive HTML documents form a web server or form local storage and render them into multimedia web pages. Html describes the structure of a web page semantically and originally included cues for the appearancs of the document.

HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects, such as interactive forms, may be embedded into semantics for text such as headings, paragraphs, lists, quotes and other items. HTML elements are delineated by tags, written using angle brackets.
